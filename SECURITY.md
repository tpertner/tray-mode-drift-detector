# Security Policy

## Supported Versions

This project is currently in early development. Security fixes will be applied to the default branch.

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, email:

- **tracylynpertner@proton.me**

Include:
- A clear description of the issue
- Steps to reproduce
- Potential impact (what could go wrong)
- Any suggested fix (optional)

I will respond as soon as possible and coordinate a responsible disclosure timeline if needed.
